Multi-Currency Wallet Using Blockchain

Introduction

This is the homepage of Multi-Currency Wallet Using Blockchain. Our project is to develop a wallet which supports the arrangement of accounts from different banks and of different types of currencies. We build a blockchain network among the partic-ipating banks to realize the on-time, money-saving and decentralized cross border money transfer.


Peocedures

The main objectives are listed as follows:
1. Creating virtual banking accounts of different currency;
2. Building a blockchain network using Hyperledger Fabric;
3. Running a trading system between different accounts using the chain;
4. Writing a mobile wallet application (at least one platform) to visualize the model.
